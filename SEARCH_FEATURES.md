# SI 파트너 검색 & 상세 페이지 (PROMPT 3 구현)

## 구현 완료된 기능

### 1. SI 파트너 검색 페이지 (/search)

**페이지 경로**: `/search`  
**접근 권한**: 공개 (로그인 불필요)

#### 핵심 기능

**✅ URL Query Params 기반 필터 관리**
- 필터 변경 시 URL에 자동 반영
- 브라우저 뒤로가기 지원
- 북마크 및 공유 가능한 검색 결과
- 페이지 새로고침 시에도 필터 상태 유지

**URL 파라미터 예시**:
```
/search?region=서울,경기&brand=UR&tag=용접&badge=true&tier=Gold&page=2
```

#### 필터 패널

**Desktop (≥1024px)**: 
- 240px 고정 좌측 사이드바
- Sticky 위치 (스크롤 시에도 고정)

**Mobile (≤1023px)**:
- "필터" 버튼으로 전체 화면 모달 오픈
- 활성 필터가 있을 때 배지 표시
- "적용하기" 버튼으로 닫기

**필터 옵션**:
1. **지역** (다중 선택)
   - 서울, 경기, 인천, 부산, 대구, 광주, 대전

2. **제조사 브랜드** (다중 선택)
   - UR, 두산로보틱스, 레인보우로보틱스, ABB, KUKA, FANUC
   - 파트너가 보유한 활성 뱃지 기준

3. **기술 역량 태그** (다중 선택)
   - 용접, 조립, 도장, 검사, 팔레타이징, 픽앤플레이스, CNC 로딩, AGV, 협동로봇, 비전검사

4. **등급** (다중 선택)
   - Diamond, Gold, Silver

5. **인증 파트너만 보기** (토글 스위치)
   - **중요**: `is_active=true` AND `expires_at > NOW()` 엄격 검증
   - 0% 미인증 오염 보장

**필터 초기화**:
- "필터 초기화" 버튼으로 모든 필터 제거
- URL도 함께 초기화

#### 검색 결과 카드

**SiPartnerCard 컴포넌트 표시 정보**:
- ✅ 회사명 + 등급 뱃지 (Diamond/Gold/Silver, 색상 구분)
- ✅ 주요 활동 지역
- ✅ 시공 성공률 (Progress Bar + 퍼센트)
  - 완료/전체 프로젝트 건수 표시
- ✅ 기술 역량 태그 (최대 5개 표시, "+N more")
- ✅ 인증 뱃지 아이콘 (활성 뱃지만)
- ✅ 평균 평점 (별점 + 리뷰 개수)
- ✅ "상세 보기" 버튼 → `/search/{siPartnerId}`

**반응형 그리드**:
- Desktop (≥1280px): 3열
- Tablet (1024-1279px): 2열
- Mobile (≤1023px): 1열 전체 너비

#### 페이지네이션

**서버 사이드 페이지네이션 시뮬레이션**:
- 페이지당 10개 결과
- 최대 5개 페이지 번호 표시
- 이전/다음 버튼
- 현재 페이지 하이라이트
- 생략 표시(...) 자동 추가
- 페이지 변경 시 상단으로 부드러운 스크롤

**접근성**:
- `aria-label="페이지네이션"`
- `aria-current="page"` (현재 페이지)
- `aria-label` (각 버튼)

#### 로딩 상태

**Skeleton 카드 (6개)**:
- 실제 카드와 동일한 레이아웃
- 애니메이션 효과
- 약 800ms 로딩 시뮬레이션

#### 빈 상태

**조건에 맞는 파트너가 없을 때**:
- 필터 아이콘
- "조건에 맞는 SI 파트너가 없습니다" 메시지
- "필터를 조정하거나 초기화하여 다시 검색해보세요" 안내
- "필터 초기화" 버튼

### 2. SI 프로필 상세 페이지 (/search/:siPartnerId)

**페이지 경로**: `/search/:siPartnerId`  
**접근 권한**: 공개 (SEO 최적화)

#### 헤더 섹션

- 회사명 + 등급 뱃지
- 주요 활동 지역
- 가입일 + 마지막 업데이트 날짜
- **"기술 리포트 PDF 다운로드" 버튼** (우측 상단)

#### 주요 메트릭 카드 (3개 가로 배치)

**1. 재무등급**
- NICE 등급 (AAA, AA, A, BBB, BB 등)
- 갱신 날짜 표시
- "운영팀 사전 업데이트 기반" 안내 문구
- 파란색 강조

**2. 시공 성공률**
- 큰 퍼센트 표시 (녹색)
- 원형 Progress Bar
- 완료/실패 프로젝트 건수
- 계산 공식: `(완료 / (완료 + 실패)) × 100`

**3. 평균 평점**
- 별점 아이콘 + 숫자
- 리뷰 개수 표시
- 노란색 별점

#### 탭 섹션

**shadcn/ui Tabs 컴포넌트 사용**:

**Tab 1: 역량 & 태그**
- 전체 기술 역량 태그 목록 (뱃지 형태)
- 프로젝트 이력:
  - 완료 프로젝트 (녹색 카드)
  - 실패 프로젝트 (빨간색 카드)

**Tab 2: 인증 뱃지**
- 뱃지 목록 (카드 형태):
  - 제조사명
  - 발급일 / 만료일
  - 활성 상태 (활성/만료됨 뱃지)
  - 만료된 뱃지는 회색 처리 + 투명도 60%
- 뱃지가 없을 때: "등록된 인증 뱃지가 없습니다"

**Tab 3: 리뷰 요약**
- `review_summary` 필드 (JSONB 안전 처리)
- 긍정적인 피드백 (녹색 섹션)
- 개선 요청 사항 (오렌지 섹션)
- 종합 평가
- 리뷰가 없을 때: "아직 리뷰 요약이 없습니다"

#### PDF 다운로드 기능

**구현 사항**:
- 버튼 클릭 → 로딩 상태 ("PDF 생성 중...")
- 2초 시뮬레이션 (실제로는 POST /api/reports/pdf)
- 성공 시 Toast 메시지

## 테스트 시나리오

### 검색 페이지 테스트

1. `/search` 접속
2. **필터 테스트**:
   - "서울" 지역 선택 → URL에 `?region=서울` 추가 확인
   - "UR" 브랜드 추가 → URL에 `&brand=UR` 추가 확인
   - "인증 파트너만 보기" 토글 → 활성 뱃지 있는 파트너만 표시 확인
   - "필터 초기화" → 모든 필터 제거 및 URL 초기화 확인
3. **페이지네이션**:
   - 페이지 2 클릭 → URL에 `&page=2` 확인
   - 상단으로 스크롤 확인
   - 브라우저 뒤로가기 → 페이지 1로 복귀 확인
4. **빈 상태**:
   - 모든 지역 + 모든 브랜드 선택 → 0건 결과 시 빈 상태 메시지 확인

### 상세 페이지 테스트

1. 검색 결과에서 "상세 보기" 클릭
2. **헤더**: 회사명, 등급 뱃지, 지역, 날짜 표시 확인
3. **메트릭 카드**: 재무등급, 성공률, 평점 표시 확인
4. **탭 전환**:
   - "역량 & 태그" → 태그 목록 + 프로젝트 이력 확인
   - "인증 뱃지" → 뱃지 목록 + 활성/만료 상태 확인
   - "리뷰 요약" → 긍정/부정 피드백 + 종합 평가 확인
5. **PDF 다운로드**:
   - 버튼 클릭 → "PDF 생성 중..." 로딩 확인
   - 2초 후 성공 Toast 확인

### 404 에러 테스트

1. `/search/invalid-id` 접속
2. "파트너를 찾을 수 없습니다" 메시지 확인
3. "검색으로 돌아가기" → `/search`로 이동 확인
